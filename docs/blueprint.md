# **App Name**: EarlyBird Signups

## Core Features:

- Interest Form: A simple form to collect user's contact information and interest.
- Google Sheets Integration: Automatically save form submissions to a designated Google Sheet.
- Logo Display: Display company logo prominently at the top of the landing page.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to convey a sense of calm and trustworthiness.
- Background color: Very light blue (#F0F8FF), nearly white, for a clean look.
- Accent color: Slightly darker blue (#77B5FE) for the call to action button to draw attention.
- Font: 'Inter', sans-serif, suitable for both headlines and body text, maintaining a clean, modern aesthetic.
- Simple, centered layout with the logo at the top, followed by a concise form. Minimal distractions.
- Use minimalistic icons to enhance visual appeal without cluttering the page.
- Subtle fade-in animations upon page load for a smooth user experience.
